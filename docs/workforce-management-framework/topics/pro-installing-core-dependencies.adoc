.Procedure

== Install Core Dependencies

[source,bash]
----
cd ./core
npm install
npm run bootstrap
----

NOTE: The Core repository is automatically cloned by the git command `npm run bootstrap`.
To work with a different branch of the Core repository, checkout in another branch.
