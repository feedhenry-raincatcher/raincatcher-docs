= Running the {WFM-RC-AppType2} Applications

//Concepts
The *{WFM-RC-NameShort}* {WFM-RC-AppType2} Apps comprise of three applications:

* _Mobile_
* _Portal_
* _Node.js Server_

For testing and demo purposes, _Mobile_ and _Portal_ are launched in the browser.
_Mobile_ can be built to run on a mobile phone.
